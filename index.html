How to Use QRit

  1. Click **'+ Add Plants to List'** to search the database and select up to **20 plants**. Optionally, add a custom URL for the QR code.
  2. Enter the **Institution Name** (optional) and select **Colors**.
  3. Click **'Generate Labels'** to display the output.
  4. Use the **'Download'** or **'Print (Browser Dialog)'** options to get your final labels.
